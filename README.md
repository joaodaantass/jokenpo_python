# 🎮 Jokenpô em Python

Um jogo simples de **Pedra, Papel e Tesoura** feito em **Python**, onde o jogador enfrenta o computador no terminal.

## 🛠 Tecnologias

* Python
* random
* emoji

## ▶️ Como executar

Instale a biblioteca:

```
pip install emoji
```

Depois execute:

```
python main.py
```

## 📚 Sobre o projeto

Projeto criado para praticar **lógica de programação, condicionais e interação com o usuário em Python**.
